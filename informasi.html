Informasi

Halaman ini berisi penjelasan mengenai website portofolio yang dibuat sebagai bagian dari tugas Ujian Akhir Semester (UAS).

Tentang Website

Website portofolio ini dibuat sebagai media untuk menampilkan identitas diri, karya desain, serta media yang telah dibuat selama proses pembelajaran.

Tujuan Pembuatan

  • Melatih pembuatan website statis
  • Menerapkan HTML, CSS, dan JavaScript dasar
  • Menyusun portofolio digital secara terstruktur

Struktur Halaman

  • Home
  • Profile
  • Project
  • Media
  • Informasi
  • Kontak

Teknologi yang Digunakan

  • HTML5
  • CSS3
  • JavaScript (Basic)
  • Canva & Affinity (Untuk Desain Visual)

Konsep Desain

Website ini menggunakan konsep desain modern dengan tampilan sederhana, fokus pada visual yang rapi, nyaman dilihat, dan mudah digunakan.

Pengembangan ke Depan

Website ini diharapkan dapat dikembangkan lebih lanjut sebagai portofolio profesional serta menjadi dasar pembelajaran UI/UX dan web development.
